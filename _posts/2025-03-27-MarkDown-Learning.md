#MarkDown 初认识

##块级元素
特征： 至少占据一行
1.标题：  #最大的标题（MarkDown 初认识）、##小一号的标题（块级元素）...
2.引用：  字符“>”加上引用的具体内容，不会显示字符‘>’  引用时想要空几行就空几行再使用">"，两个引用会合并且保留空格
>加上后面具体内容


>空两行
3.有序列表： 1.标题 2.引用 3.有序列表 带有数字标号
4.无序列表： 字符“+、-、*”三个都一样，空格加上内容

+ 无序列表1
- 无序列表2
* 无序列表3

5.任务列表：   字符"-[ ] 吃饭"

-[ ] 吃饭
-[ ] 睡觉
-[ ] 打豆豆

6.代码块：		三个'''c加上语言
'''c
int main(){
    return 0;
}

|姓名|年龄|成绩|

一键三连[^三连]
